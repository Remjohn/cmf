# The Viral Scoring System: Sacred Return Arc

## Core Identity
**Arc Type:** The Sacred Return (Sonic Arc #11)
**Goal:** Integration / The Gift / Full Circle
**Emotional Frequency:** Departure → Trial → Return → Gift

---

## 1. The Specificity Ladder (The Scar)
**Question:** How do we know you went to battle?
*We need physical proof of the journey.*

| Score | Level | Description | Example |
|-------|-------|-------------|---------|
| **10/10** | **The Artifact** | Physical object or scar. | "I still have the boots I wore that day." |
| **8-9/10** | **The Change** | Specific behavioral shift. | "I stopped locking my door." |
| **6-7/10** | **The Memory** | "I remember the fear." |
| **4-5/10** | **The Lesson** | "I learned a lot." |
| **0-3/10** | **The Vague** | "It was hard." |

**Validation Rule:** IF Cluster is **SR2 (The Trials)**, Specificity MUST be ≥8/10.

---

## 2. Setup Validation (The Deployment)
**Question:** Did they actually leave?

| Score | Level | Metric |
|-------|-------|--------|
| **10/10** | **The Door** | Describe walking out/leaving. |
| **7-9/10** | **The Decision** | "I booked the ticket." |
| **4-6/10** | **The Plan** | "I wanted to go." |
| **0-3/10** | **The Dream** | "I imagined it." (Fails Setup). |

**Validation Rule:** IF Cluster is **SR1 (Old World)**, Score MUST be ≥7/10.

---

## 3. Vulnerability Hierarchy (The Change)
**Question:** Is the "Old Self" dead?

| Score | Level | Description |
|-------|-------|-------------|
| **10/10** | **The Stranger** | "I didn't recognize myself." |
| **8-9/10** | **The Difference** | "I don't react the same way." |
| **6-7/10** | **The Growth** | "I'm better now." |
| **0-5/10** | **The Same** | "I'm back to normal." (Fails Return). |

**Validation Rule:** The Return must show ALTERATION.
