## Scoring Table (0-10)

| Score | Description | Effect on Combined Score |
|-------|-------------|--------------------------|
| **10** | **Holographic:** The quote encapsulates the WHOLE Frame (Problem + Mechanism + Result) in one sentence. | **Multiplier: 1.5x** |
| **8-9** | **Strong Support:** The quote directly addresses a key pillar of the Frame (e.g., specific Problem or specific Result). | **Multiplier: 1.2x** |
| **5-7** | **Neutral/Tangential:** Good content, but could belong to any interview. Generic wisdom. | **Multiplier: 1.0x** |
| **3-4** | **Distraction:** Interesting sidebar that pulls focus AWAY from the main arc. | **Multiplier: 0.5x** |
| **0-2** | **Contradiction:** Active disagreement with the Frame or confusingly off-brand. | **Multiplier: 0.0x (DISCARD)** |
